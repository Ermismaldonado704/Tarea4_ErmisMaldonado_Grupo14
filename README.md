# Tienda de Productos Vegetales - MongoDB

## Descripción

Este proyecto corresponde a la implementación de una base de datos NoSQL utilizando MongoDB para la gestión de productos de una tienda de vegetales.

La base de datos permite almacenar información sobre frutas, verduras, hortalizas y tubérculos, facilitando operaciones de consulta, inserción, actualización, eliminación y análisis de datos.

## Base de datos

**Nombre:** tienda_vegetales

**Colección principal:** productos

## Campos utilizados

- nombre
- categoria
- precio
- stock
- proveedor

## Operaciones implementadas

- Inserción de documentos
- Consulta de datos
- Actualización de documentos
- Eliminación de registros
- Filtros de búsqueda
- Conteo de documentos
- Suma de precios
- Promedio de precios
- Agrupación por categoría

## Tecnologías utilizadas

- MongoDB
- Mongo Shell (mongosh)
- GitHub
